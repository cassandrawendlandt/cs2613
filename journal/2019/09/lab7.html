What new concepts (if any) did you learn about in this lab?

Macros

In the lab we discussed the construction of programs using macros. To define a macro in racket we use define-syntax-rule. This is basically a function that does not evaluate the arguments. This seems like it would be helpful in a case where you want to evaluate math functions or different boolean statements.

Pattern Matching

The next new thing that was covered in the lab was match. This is a function where you specify a pattern to match against the data instead of specifying a test for each case. This seems like a very helpful when using a pattern against a list.
